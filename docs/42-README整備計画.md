## 5. 書誌取得 GIF

ISBN スキャン → 書誌・書影が埋まる流れは書き込みを伴うので、ライブ
デモではなく GIF で見せる (41-QR-search/docs/08 Q3-2)。4 つの中で
唯一、実機カメラ or 入力注入が要る = 一番手がかかる。

- 撮影環境: **ローカル** (デモには API キーが無いので撮れない)。
  ローカル .env に楽天/Yahoo キーを入れ、実在の ISBN で撮る
- スキャン部分はカメラ実機が理想。カメラ無しなら file input への
  画像注入で代替できる (local-e2e メモの手法)
- 録画 → GIF 変換: ffmpeg (palettegen/paletteuse) で 10〜15 秒・
  幅 480px・**3MB 以下**を目標。厳しければ動画 (mp4) を README に
  リンクする形へ落とす
- 置き場: `docs/assets/isbn-lookup.gif`。README のデモ節の下に
  「書誌の自動取得 (デモでは無効)」として置く
