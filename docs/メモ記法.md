# メモ記法

memo 欄で使える Markdown 記法のメモ(2026-07-15 時点)。
実装は `src/components/MarkdownView.tsx` と
`MermaidDiagram.tsx`。

## 基本(Markdown)

- GFM 対応: 表・タスクリスト・打消し線・裸 URL の自動リンク
  (remark-gfm)
- 単一改行はそのまま改行として表示される(remark-breaks)
- 生 HTML は無効(react-markdown のデフォルト + rehype-sanitize)。
  `<img width="...">` のような HTML タグは書けない
- 本文は最大 10,000 文字

## 画像

- エディタの添付ボタン・貼り付け・ドロップでアップロードすると
  `![](/api/images/<uuid>.png)` が本文に挿入される
- 対応形式: png / jpeg / gif / webp、1 ファイル 10MB まで
- 保存先: `UPLOAD_DIR`(compose では named volume `uploads-data`)
- 画像はクリックで拡大表示される(もう一度クリックか Esc で閉じる)

### 幅指定(独自記法)

alt の末尾に `|数字` を付けると表示幅(px)を指定できる(Obsidian 風)。

```text
![|200](/api/images/xxxx.png)          ← 幅 200px、alt なし
![スクショ|200](/api/images/xxxx.png)  ← 幅 200px、alt「スクショ」
```

- 高さは縦横比を保って自動で決まる
- 末尾が数字でない `|` は通常の alt として扱う(例: `![a|b](...)`)

## 数式(KaTeX)

`$...$` で囲むとインライン数式、`$$` だけの行で囲むとブロック数式として
KaTeX で描画される(remark-math + rehype-katex、サーバー側で HTML に変換)。

```text
インライン: 質量エネルギーは $E = mc^2$ で表せる

ブロック:
$$
\int_0^1 x^2 \, dx = \frac{1}{3}
$$
```

- 使える記法は KaTeX がサポートする LaTeX コマンドの範囲
  ([https://katex.org/docs/supported](https://katex.org/docs/supported))
- 通貨表記など数式にしたくない `$` は `\$` とエスケープする。
  同じ段落に `$` が 2 つあると、挟まれた文字列が数式として
  解釈されることがある
- 構文エラーの数式は、書いた内容がそのまま赤字で表示される

## mermaid

` ```mermaid ` フェンスを書くと図として描画される
(mermaid v11.16 を動的 import)。

````markdown
```mermaid
flowchart TD
    A --> B
```
````

### サポートする図種

mermaid v11.16 本体に含まれる図はすべて描画できる
(外部プラグインが必要な zenuml を除く)。主なもの:

| 記法(先頭行) | 図 |
| --- | --- |
| `flowchart` / `graph` | フローチャート |
| `sequenceDiagram` | シーケンス図 |
| `classDiagram` | クラス図 |
| `stateDiagram-v2` | 状態遷移図 |
| `erDiagram` | ER 図 |
| `gantt` | ガントチャート |
| `pie` | 円グラフ |
| `gitGraph` | git ブランチ図 |
| `mindmap` | マインドマップ |
| `timeline` | 年表 |
| `quadrantChart` | 4 象限チャート |
| `xychart-beta` | 折れ線・棒グラフ |
| `sankey-beta` | サンキー図 |
| `block-beta` | ブロック図 |
| `packet-beta` | パケット図 |
| `kanban` | かんばんボード |
| `architecture-beta` | アーキテクチャ図 |
| `journey` | ユーザージャーニー |
| `requirementDiagram` | 要求図 |
| `C4Context` など | C4 モデル図 |

### サイズ調整

図の先頭に init ディレクティブを書くとフォントサイズを変えられる
(フォントに合わせて図全体も縮む)。

```text
%%{init: {"themeVariables": {"fontSize": "12px"}}}%%
```

図はコンテナ幅が上限(useMaxWidth: true)なので、
自然サイズより大きく引き伸ばされることはない。

### 制限(securityLevel: "strict")

- `click` によるリンク・コールバックは無効
- ラベル内の HTML はエスケープされる
- `securityLevel` や `maxTextSize` などの secure 設定は
  init ディレクティブから変更できない
